## 🔍 1.3 승진 프로세스 페이지 (promotion-process.html)

### 기능
교원인사규정 제3장을 기반으로 승진 프로세스를 시각화

### 주요 표시 내용

#### A. 승진 자격 요건
```
┌─────────────────────────────────────────────┐
│ 승진 소요 연한                                │
├─────────────────────────────────────────────┤
│ • 2012.3.1 이후 신규임용 정년트랙 교원       │
│   - 조교수 → 부교수: 6년                     │
│   - 부교수 → 교수: 7년                       │
│                                             │
│ • 2012.2.28 이전 신규임용 정년트랙 교원      │
│   - 조교수 → 부교수: 4년                     │
│   - 부교수 → 교수: 5년                       │
│                                             │
│ • 비정년트랙 교원                            │
│   - 조교수 → 부교수: 6년                     │
└─────────────────────────────────────────────┘
```

#### B. 승진 일정 타임라인
```
┌──────────────────────────────────────────────────────────┐
│  4월 1일 승진의 경우                                        │
├──────────────────────────────────────────────────────────┤
│  [전년 12월] → [1월] → [2월 말] → [3월] → [4월 1일]        │
│      ↓          ↓        ↓         ↓         ↓            │
│   대상자      서류     제출      심사      승진           │
│   파악       준비     마감      완료      발령           │
└──────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│  10월 1일 승진의 경우                                       │
├──────────────────────────────────────────────────────────┤
│  [6월] → [7월] → [8월 말] → [9월] → [10월 1일]             │
│   ↓      ↓        ↓         ↓         ↓                   │
│  대상자  서류    제출      심사      승진                  │
│  파악   준비    마감      완료      발령                  │
└──────────────────────────────────────────────────────────┘
```

#### C. 승진 제한 사유
- 징계처분·직위해제 또는 휴직 중인 자
- 징계처분 집행 종료 후 일정 기간 미경과자
  - 정직: 18개월
  - 감봉: 12개월
  - 견책: 6개월

#### D. 필요 서류
- 교원계약임용신청서 (별표5)
- 교원업적평가표
- 연구실적물
- 기타 심의 필요 서류

